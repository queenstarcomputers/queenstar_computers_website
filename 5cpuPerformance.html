Factors that Determine Performance/Speed of CPU

Queenstar Computers: 15 Dec, 2023 | Category>> Hardware

The Central Processing Unit (CPU) is the main part of a computer system like the brain of a human being. The OS loads program instructions and data into the main memory (RAM) and the CPU interprets the instructions in the program and executes them one by one by fetching these instructions from the main memory. The CPU is mainly comprised of the control unit (CU), the arithmetic/logic unit (ALU), the registers, and the system clock. The major functions of the CPU include: Processing data held in main memory (RAM); controlling transmission of data from input devices to main memory; and controlling transmission of information from main memory to output devices.

In order for the CPU to perform in an efficient manner there are a variety of factors that must be considered as follows:

1) Clock speed

Clock speed is the speed at which the CPU can carry out instructions. The speed of CPU is known as Computer Speed. The clock speed indicates the speed of CPU and is measured by Hertz (Hz) which represent a CPU cycle (one cycle per second). With every tick of the clock, the CPU fetches and executes one instruction. To synchronize/ coordinate the various steps carried out during the fetch-execute cycle, the CPU has an internal clock that generates regularly timed pulses. All the processor activities, such as fetching an instruction, reading data into the memory register, etc. must begin on a clock pulse/ tick, (although some activities take more than one clock pulse to complete). CPU clock is only used on the processor chip itself.

The higher the clock speed a CPU has, the faster it can process instructions.

2) Core/ CPU core

This is the "brain" of a CPU. It receives instructions and performs calculations, or operations, to satisfy those instructions. Traditionally, a CPU is made up of a processor with a single core. A CPU can have multiple cores (Multi-core processors). Multi-core processor is a computer processor with 2 or more separate processing units (CPUs)/ cores. The higher the core count a processor has, the higher the performance of a CPU.

Dual-core processor Has 2 cores (is like having 2 processors in one/ has 2 central processing units on the CPU chip)
Quad-core processor Has 4 cores (is like having 4 processors in one/ has 4 central processing units on the CPU chip)
Hexa-core processor Has 6 cores (is like having 6 processors in one/ has 6 central processing units on the CPU chip)
Octa-core processor Has 8 cores (is like having 8 processors in one/ has 8 central processing units on the CPU chip)

3) Cache memory

A cache is a tiny block of very fast memory built into the CPU used to store the most commonly used instructions and data so that they are close at hand when required by the CPU. It is the 2nd fastest after registers. The bigger the cache, the more quickly the commonly used instructions and data can be brought into the processor and used. The data stored in a cache can be values that have been computed earlier or duplicates of original values stored elsewhere. CPU uses cache memory to store instructions that are repeatedly required to run programs, improving overall system speed. If the cache memory gets full, it slows down the computer's performance (lagging).to solve this the cache is cleared regularly hence increasing the performance of the computer system. This can be done by clearing your browser history or temporary internet files.

4) Word size

A computer with a word size of 64 bit can process information at a faster rate than one with a word size of 32. The 64-bit will enable the processor to access more RAM, run applications more efficiently, and also run both 32-bit and 64-bit programs.

NB:A word is a group of fixed numbers of processed as a unit, which varies from computer to computer. The length of a computer word is called word size or word length i.e. the number of bits that the processor can process simultaneously. The word size may be as small as 8 bits or may be as long as 96 bits.

NBNibble= 4 bits; Byte = 8 bits; Word = 32 bits (4 bytes); Double word = 64 bits (8 bytes).

5) Bus size

The wider the data bus, the more and faster the flow of data on the data bus and thus better system performance. The wider the address bus, the more address locations accessed per time unit (second) hence a faster-running computer.

6) Memory access time

It is the amount of time it takes the processor to read or store data, instructions, and information from memory. The higher the access time the faster the flow of data hence higher performance.

7) Architecture

ISA (Industry Standard Architecture): carries data along 16 lines (16-bit bus) whereas EISA (Enhanced Industry Standard Architecture) or MCA (Micro Channel Architecture) carry data along 32 lines (32-bit bus) hence faster.
